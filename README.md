# SWP391_BE

## Overview
SWP391_BE is a backend solution developed using C# 12 and .NET 8. It is structured to support scalable web API development and follows modern software engineering practices. The solution is organized into multiple projects for clear separation of concerns, including API, services, repositories, and contracts.

## Projects
- **SWP391.WebAPI**: Exposes RESTful endpoints for client applications.
- **SWP391.Services**: Contains business logic and service implementations.
- **SWP391.Repositories**: Manages data access and persistence.
- **SWP391.Contracts**: Defines shared interfaces and data models used across the solution.

## Getting Started
To run the solution:
1. Ensure you have .NET 8 SDK installed.
2. Restore dependencies using your preferred package manager.
3. Build and run the `SWP391.WebAPI` project.
